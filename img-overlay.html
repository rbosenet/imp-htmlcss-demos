Original Image using img tag 420px x 682px

Image : width 200px Image doesnt get cropped

Image : width 600px Image doesnt get cropped

Bg Images will get cropped if bg size set to cover and bg img apect raio is different from conatiner aspect ratio

Default bg image . This is 500px x 300px container with 420px x 682px bg img. by default left corner of img coincicdes with left corner of the cotainer div and it repeats. Notice the image gets cropped vertically as the container height (300px) is smaller than the image height 682px

This div is same width as above but with bg-img set to cover and no height but aspect ratio same as image aspect ratio in which case nothing gets cropped. This is the part to remember : for bg-img set to cover bg- Images will get cropped if the container aspect ratio doesnt match bg-img aspect ratio .
